# English Kids Learning 🎨📚🇺🇸

Aplicación web educativa e interactiva desarrollada en React para apoyar el aprendizaje de inglés básico en niños de 4 a 7 años.

El proyecto está orientado a estudiantes de preescolar y primaria, permitiendo aprender vocabulario mediante videos, imágenes, audios, actividades interactivas y evaluaciones gamificadas.

---

## 🎯 Objetivo

Facilitar el aprendizaje de conceptos básicos de inglés de forma divertida, visual e intuitiva, utilizando recursos multimedia adaptados a la edad de los estudiantes.

La aplicación está diseñada para ser utilizada tanto en instituciones educativas como en el hogar.

---

## 👥 Usuarios

### 🧑‍🎓 Estudiante

Los estudiantes pueden:

* Aprender nuevos temas mediante lecciones interactivas.
* Escuchar pronunciaciones en inglés y español.
* Ver videos educativos.
* Realizar actividades de práctica.
* Presentar evaluaciones.
* Obtener calificaciones.
* Ganar estrellas e insignias.
* Consultar su progreso.

### 👩‍🏫 Profesor

Los docentes pueden:

* Configurar los temas que aparecerán en las evaluaciones.
* Consultar estadísticas básicas.
* Reiniciar la información almacenada en el dispositivo.
* Gestionar el entorno de aprendizaje para cada grupo de estudiantes.

## 📝 Sistema de Evaluación

La aplicación genera preguntas aleatorias basadas en los temas seleccionados por el profesor.

### Tipos de actividades

* Selección múltiple.
* Identificación visual.
* Relación imagen-palabra.
* Escuchar y seleccionar.
* Arrastrar y soltar.

### Escala de Calificación

| Nota      | Desempeño     |
| --------- | ------------- |
| 5.0       | Excelente     |
| 4.5 - 4.9 | Superior      |
| 4.0 - 4.4 | Alto          |
| 3.5 - 3.9 | Aprobado      |
| 1.0 - 3.4 | Debe reforzar |

La nota mínima aprobatoria es 3.5.

## 🏆 Gamificación

El aprendizaje se fortalece mediante un sistema de recompensas.

### Estrellas

Se obtienen al responder correctamente las actividades.

### Insignias

* Explorador de Colores
* Maestro de Figuras
* Experto en Útiles Escolares
* Pequeño Experto en Inglés

---

## 📊 Seguimiento del Progreso

La aplicación almacena localmente:

* Nombre del estudiante.
* Temas completados.
* Evaluaciones realizadas.
* Calificaciones obtenidas.
* Promedio general.
* Insignias ganadas.
* Estrellas acumuladas.

---

## 🔒 Almacenamiento Local

La información se guarda utilizando Local Storage del navegador.

No se requiere:

* Base de datos.
* Servidor backend.
* Registro de usuarios.
* Conexión permanente a internet.

## 🚀 Instalación

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Ingresar al proyecto:

```bash
cd english-kids-learning
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```

Generar versión de producción:

```bash
npm run build
```
